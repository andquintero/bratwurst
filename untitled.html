1. Introduction

Why another NMF package?
  • Implemented on tensorflow, allows better scalability and processing speed. Including processing of single-cell data sets, with thousands of cells.
  • This package compiles several NMF variants, allowing the user to try multiple variations if the algorithms without installing multiple packages.
  • The package is available to install from https://github.com/hdsu-bioquant/bratwurst, and is also ready to use from the docker image hdsu/bratwurst_example.
